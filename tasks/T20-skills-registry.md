# T20 · 技能注册表（符号工具 + 感知-运动技能、示范学习雏形）

- 里程碑：M2.8
- 依赖：T09、T13、T17
- PRD 对应：FR-9、§6.5.1 技能形态巨变
- 状态：⬜ 未开始

## 目标
技能在感知式 Agent 里分裂为两类并存：**符号工具技能**（文本 schema 的函数/API）与**感知-运动技能**（"感知→行动"过程性策略，如"登录这个 App""关掉这个弹窗"，靠示范/经验习得，落在程序性记忆）。建一个能容纳这条谱系的注册表。

## 交付物
- [ ] `src/opensensorium/skills/registry.py`：技能注册表，走与 Effector 相同的能力协商。
- [ ] `src/opensensorium/skills/symbolic.py`：符号工具技能（内置：搜索、文件、通知等；复杂者转交 T16 Delegate）。
- [ ] `src/opensensorium/skills/perceptual_motor.py`：感知-运动技能——"看→动→验证"策略（复用 T15 校验闭环），存取程序性记忆（T17）。
- [ ] 技能习得雏形：成功的感知-行动轨迹可**巩固为可复用的感知-运动技能**（FR-9.3，示范学习）。
- [ ] 技能同样过 T14 Safety 层。
- [ ] 测试：`tests/test_skills.py`。

## 分工（Agent 做什么 / 人做什么）
- **Agent**：实现注册表、两类技能基类、示范→巩固管线、单元/集成测试。
- **人**：示范一次真实感知-运动技能（如登录某 App）供习得；确认复用效果。

## 验收标准（可验证）
- [ ] `tests/test_skills.py`：两类技能都能注册与调用；符号技能走能力协商；感知-运动技能能保存/复用。
- [ ] 集成：**示范一次**某流程后，能把它复用为一个感知-运动技能再次执行成功。

> ⚠️ 需人工验证：示范学习效果由你实测。
