# T21 · 主动性三触发源统一仲裁 + 显著性调优（不烦人）

- 里程碑：M3
- 依赖：T12、T18
- PRD 对应：§3.7 三触发源、FR-2.3、§4.2 场景 2、§4.3
- 状态：⬜ 未开始

## 目标
主动陪伴/观察式助手：把**感知触发（salience）+ 时间触发（schedule）+ 事件触发（event）**三源统一进认知核仲裁"此刻做什么"，并把显著性阈值调到"主动但不烦"，得许可即动手。

## 交付物
- [ ] `src/opensensorium/core/arbiter.py`：三触发源统一仲裁器（优先级 + 显著性 + 当前任务）。
- [ ] 主动提示策略：卡在报错时主动、恰当地出手/提示/改；得许可即动手（观察与操作连续）。
- [ ] 显著性阈值调优面板/配置（防话痨旋钮）。
- [ ] 测试：`tests/test_arbiter.py`（三源并发时的仲裁选择）。

## 分工（Agent 做什么 / 人做什么）
- **Agent**：实现仲裁器、策略、单元/集成测试。
- **人**：长时间真实陪伴体验，调显著性阈值到"主动但不烦"，确认"得许可即动手"的分寸。

## 验收标准（可验证）
- [ ] `tests/test_arbiter.py`：三源同时到达时按优先级/显著性正确选中一个动作。
- [ ] 集成：模拟"卡在报错"场景，它主动提示并在得许可后动手。
- [ ] 阈值可调，调高后主动频率下降（可量化）。

> ⚠️ 需人工验证："主动但不烦"体感与出手分寸由你长时间实测（建议多轮调优）。
