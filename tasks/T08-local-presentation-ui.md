## 目标
提供本地叠加式呈现面 UI，建立信任："它现在在看什么、准备做什么、做了什么"。该 UI 默认不进入采集（复用 T06 排除）。

## 交付物
- [ ] `src/opensensorium/endpoint/windows/ui.py`：叠加浮层窗口（PySide6/Qt 或轻量方案）。
- [ ] 显示：当前被观察面缩略/标注（含当前视觉源）、结论、动作日志、委派状态、当前任务。
- [ ] self-view 与浮层默认调用 T06 的排除（FR-5.3）。
- [ ] 订阅 T04 总线渲染实时状态。
- [ ] 测试：UI 逻辑层可测部分 `tests/test_ui_state.py`（状态模型与总线订阅），渲染部分人工验证。

## 实现要点
- UI 只读呈现，不承担控制逻辑；控制指令另走控制面。
- 缩略图来自当前活动源帧（T05/T07）。

## 分工（Agent 做什么 / 人做什么）
- **Agent**：实现 UI 状态模型 + 浮层渲染 + 状态层测试。
- **人**：确认渲染观感与"可见其所见"的直观性。

- [ ] 浮层默认不进入采集帧（复用 T06 验证）。
